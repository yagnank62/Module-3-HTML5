< section > , < aside > , < details > , < dialog > , < figcaption > , < mark > , < summary > , < time >

How to embed audio and video in a webpage?

Add < audio > and < video > elements to the page; make them display the default browser controls. Give both of them < source > elements so that browsers will find the audio format they support best and load it. These should include type attributes.

Syntax of audio tag:
< audio >
< source src="file_name" type="audio/mp3" >
< /audio >

Attributes of audio tag are: autoplay, controls, loop, muted, preload, src. Audio fileformat is MP3, WAV, OGG.

Syntax of video tag:
< video >
< source src="file_name" type="video/mp4" >
< /video >

Attributes of video tag are: autoplay, controls, loop, muted, poster, preload, src, width, height. Video fileformat is MP4, WebM, OGG.

Semantic element in HTML5?

A semantic element clearly describes its meaning to both the browser and the developer.
Examples of non-semantic elements: < div > and < span > - Tells nothing about its content.
Examples of semantic elements: < form >, < table >, and < article > - Clearly defines its content.

Semantic elements in html:

article, aside, details, figcaption, figure, footer, header, main, nav, mark, section, summary, time.

Unlike raster image (Ex .jpg, .gif, .png, etc.), a vector image can be scaled up or down to any extent without losing the image quality. Canvas: The HTML element is used to draw graphics on the fly, via scripting (usually JavaScript). The element is only a container for graphics. You must use a script to actually draw the graphics. Canvas has several methods for drawing paths, boxes, circles, text, and adding images.
